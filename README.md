# 業務進捗報告

業務委託向けの業務進捗・活動報告アプリです。

## ログインID・パスワードの管理方法

**スプレッドシート連携は不要です。** アプリ内の「メンバー管理」画面で管理します。

### 推奨: 管理画面で追加（管理者のみ）

1. ログインIDが **admin** のアカウントでログインする
2. ヘッダーの **「メンバー管理」** をクリックして `/admin` を開く
3. 「メンバーを追加」フォームで **ログインID・表示名・初期パスワード** を入力して追加
4. 追加したログインIDとパスワードを、該当メンバーに共有する

データはサーバーの `data/users.json` に保存され、各メンバーの稼働データは `data/records/` 内でユーザーごとに分けて保存されます。

### 別の方法: スクリプトで一括登録

初期メンバーをまとめて登録したい場合は、`scripts/seed-users.js` の `initialUsers` を編集してから以下を実行します。

```bash
node scripts/seed-users.js
```

※ 実行すると `data/users.json` が上書きされます。既存メンバーも含めて記載してください。

## 開発の起動

```bash
npm install
npm run dev
```

ブラウザで http://localhost:3000 を開き、ログインIDとパスワードでログインしてください。

## 初期管理者

- ログインID: **admin**
- パスワード: **admin123**

（本番運用時は必ずパスワードを変更するか、管理画面で新規管理者を追加してから運用してください。）
